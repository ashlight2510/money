# 🛡️ 내 자산 방탄 지수 테스트

**경제위기 와도 내 돈은 얼마나 버틸까?**

10문항으로 알아보는 나의 자산 방어력을 측정하는 바이럴형 심리 테스트 서비스입니다.

## 📋 프로젝트 소개

부자들의 '자산 지키려는 공포 심리'를 가볍고 재밌게 풀어내는 광고 친화형 테스트입니다.

- **포지션**: 게임/심리테스트 느낌 (절대 전문 조언 아님)
- **목적**: 재테크 앱, 가계부 앱, 금융 콘텐츠 등의 광고 수익 최적화
- **타겟**: 자산 관리에 관심 있는 20~40대

## 🎯 주요 기능

1. **10문항 설문 조사**
   - 소득, 자산 구성, 비상금, 보험, 대출, 저축률 등 10가지 질문

2. **3가지 지표 측정**
   - **안정성 (Defense)**: 비상금, 보험, 분산, 현금 비율
   - **위험노출 (Risk Exposure)**: 코인/주식 편중, 빚, 소득 불안정
   - **건전성 (Fundamentals)**: 저축률, 기본 재무습관

3. **방탄지수 산출 (0~100점)**
   - 방어력 × 0.45 + 기초체력 × 0.35 + (100 - 위험노출) × 0.20

4. **3가지 투자자 타입 분류**
   - **공포형 디펜스 투자자**: 현금 비중 높음, 위기 회피 성향 강함
   - **현실형 밸런스 투자자**: defense·risk·fundamentals 비중 균형
   - **공격형 리스크 테이커**: 주식·코인 초과 투자, 변동성 높음

5. **맞춤형 조언 제공**
   - 각 타입별 라이트한 투자 가이드
   - 자산 관리 기본 가이드

6. **결과 공유 기능**
   - 카카오톡 공유
   - 링크 복사

## 📁 파일 구조

```
money/
├── index.html          # 메인 질문 페이지
├── result.html         # 결과 페이지
├── style.css           # 스타일시트
├── script.js           # 메인 로직 (점수 계산, 결과 표시)
├── questions.js        # 질문 데이터
└── README.md           # 프로젝트 설명서
```

## 🚀 사용 방법

1. **로컬에서 실행**
   ```bash
   # 간단한 HTTP 서버 실행 (Python 3)
   python -m http.server 8000
   
   # 또는 Node.js http-server 사용
   npx http-server
   ```

2. **브라우저에서 접속**
   ```
   http://localhost:8000
   ```

3. **배포**
   - GitHub Pages, Netlify, Vercel 등 정적 호스팅 서비스에 배포 가능
   - 단일 페이지 애플리케이션이므로 별도의 빌드 과정 불필요

## 🎨 커스터마이징

### 질문 수정
`questions.js` 파일에서 질문과 선택지를 수정할 수 있습니다.

### 점수 계산 로직 수정
`script.js`의 `calculateResult()` 함수에서 점수 계산 방식을 변경할 수 있습니다.

### 스타일 수정
`style.css`에서 색상, 폰트, 레이아웃 등을 커스터마이징할 수 있습니다.

### 광고 슬롯 추가
`result.html` 또는 `script.js`의 `displayResult()` 함수에서 광고 슬롯을 수정할 수 있습니다.
